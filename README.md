# 多邻国用户评论情感分析器

基于大模型的用户评论自动分析工具，上传 CSV 自动提取评论、用户名、评分，调用 DeepSeek API 分析情感，输出统计结果和可视化图表。

**核心价值**：产品经理每周5小时的评论筛选时间 → 1小时


## 模型评估结果（93条测试集）

| 指标 | 结果 |
|------|------|
| 精确率 | 96.6% |
| 召回率 | 100% |
| TP | 56 |
| FP | 2 |
| FN | 0 |

- 漏报比误报更严重，所以优先保证召回率
- 2条误报均为用户“抱怨”被误判为“差评”，后续优化Prompt区分


## 分析结果（92条真实评论）

**情感分布：**
- 负面：57条（62.0%）
- 正面：33条（35.9%）
- 中性：2条（2.1%）

**评分分布：**
- 5星：35条
- 4星：10条
- 3星：7条
- 2星：11条
- 1星：29条

**紧急问题TOP5：**
1. 会员扣费纠纷、无法退款（紧急度5）
2. 卡顿、闪退、加载失败（紧急度4）
3. 国际象棋最后一步卡死（紧急度4）
4. 希望增加请假功能（紧急度3）
5. 评分低但未明确原因（紧急度2）
