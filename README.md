# 多邻国用户评论情感分析器

## 项目简介

一款基于大模型的用户评论自动分析工具。上传CSV文件，自动提取评论、用户名、评分，调用DeepSeek API进行情感分析，输出统计结果、紧急问题排序和可视化图表。

## 核心功能

| 功能 | 说明 |
|------|------|
| 自动提取 | 自动识别CSV中的评论内容、用户名、评分（支持`内容/作者/评级`或`text/userName/score`格式） |
| 情感分析 | 调用DeepSeek API，判断每条评论的情感（正面/负面/中性） |
| 问题提取 | 自动总结每条评论的核心问题 |
| 紧急度评分 | 1-5分，标记问题的紧急程度 |
| 统计输出 | 情感分布、评分分布、紧急问题TOP5 |
| 可视化 | 自动生成情感分布饼图和评分分布柱状图 |

## 模型评估结果

基于93条人工标注的测试集：

| 指标 | 结果 |
|------|------|
| 精确率 (Precision) | **96.6%** |
| 召回率 (Recall) | **100%** |
| TP (抓对差评) | 56 |
| FP (误报) | 2 |
| FN (漏报) | 0 |
| TN (正确放过) | 35 |

### 错误分析
- **误报案例**：用户提建议（如"价格太贵了，便宜点就好了"），AI判为负面
- **改进方向**：优化Prompt，区分"抱怨"和"正式差评"
